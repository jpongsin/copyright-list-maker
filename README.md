Updated 15 Sept 2025

About
This small program essentially makes a fixed number of titles
that have been either renewed or not renewed.

There is no validation, if not little validation for the attributes.
You can view a local list that has been formatted...
And you can delete a local list.

If you set number of titles to be made, it will be overwritten.
If you create a local list after first creating a list,
it will warn you to overwrite.

This runs primarily on console terminal.

The source comes with a makefile that can be compiled automatically
and you can run it on any program.

More features might be added at some point, but this is mostly
an exercise.
