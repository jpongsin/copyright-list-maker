COPYRIGHT RENEWAL LIST CREATOR

Jonathon Pongsin

Updated 15 Sept 2025

ABOUT

This small program runs through a console terminal
and makes a number of titles in a list
that indicate if each titles have been renewed. 

You can:
Set the number of titles in a list,
Input the attributes of each title in a list,
View the list, and
Delete the list.

There is no validation, if not little validation for the attributes.
If you set number of titles to be made, it will be overwritten.
If you create a local list after first creating a list,
it will warn you if you would like to overwrite.

The source comes with a makefile that can be compiled automatically
by typing "make"
and you can run it on any program, e.g. "./CopyrightList"

More features might be added at some point, 
but this is mostly an exercise or demonstration.
